# Vindicate – Blockchain Credential Verification

### Problem
Current skill verification is centralized and often falsified.
Employers waste time verifying resumes & certificates.

### Solution
A blockchain-based system where institutions issue **verifiable credentials**.
Employers verify instantly on-chain.

### Features
- Blockchain-based skill proof.
- Issuers → upload credentials (hash stored on-chain).
- Students → own their credentials.
- Employers → verify with one click.
- Public Explorer for transparency.

### Demo Flow
1. Issuer issues credential → stored on blockchain.
2. Student sees it in dashboard.
3. Employer verifies via credential ID/hash.

### Tech Stack
- Solidity + Hardhat
- React + Tailwind
- Mock IPFS
- Optional Express.js backend
